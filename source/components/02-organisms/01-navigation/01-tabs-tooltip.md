#### Description

Finna main tabs component with tooltips. Can be used for navigations and for showing/hiding content on a page. Turns into a dropdown menu on mobile devices.

#### Properties

| Prop         | Type      | Required |
| ------------ | --------- | -------- |
| menuClass    | `String`  | No       |
| tablistLabel | `String`  | Yes      |
| tabs         | `Object[]`| Yes      |
| iconLabel    | `String`  | Yes      |
| submit       | `String`  | Yes      |
